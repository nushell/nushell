# build_string
Builds a string from the arguments.

## Usage
```shell
> build_string ...args {flags} 
 ```

## Parameters
* ...args: all values to form into the string

## Flags
* -h, --help: Display this help message

## Examples
  Builds a string from a string and a number, without spaces between them
```shell
> build_string 'foo' 3
 ```
