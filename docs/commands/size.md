# size

This commands gives word count statistics on any text.

## Examples -

```shell
> open lalala.txt | size
━━━━━━━┯━━━━━━━┯━━━━━━━┯━━━━━━━━━━━━
 lines │ words │ chars │ max length 
───────┼───────┼───────┼────────────
     4 │    10 │    72 │         72 
━━━━━━━┷━━━━━━━┷━━━━━━━┷━━━━━━━━━━━━
> open the_mysterious_affair_at_styles.txt | size
━━━━━━━┯━━━━━━━┯━━━━━━━━┯━━━━━━━━━━━━
 lines │ words │ chars  │ max length 
───────┼───────┼────────┼────────────
```
