# size

This commands gives word count statistics on any text.

## Examples

```shell
> open lalala.txt | size
━━━━━━━┯━━━━━━━┯━━━━━━━┯━━━━━━━━━━━━
 lines │ words │ chars │ bytes
───────┼───────┼───────┼────────────
     4 │    10 │    72 │         72
━━━━━━━┷━━━━━━━┷━━━━━━━┷━━━━━━━━━━━━
```
