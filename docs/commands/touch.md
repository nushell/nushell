Creates one or more files.

## Signature

```> touch (filename) ...rest```

## Parameters

 -  `filename`: the path of the file you want to create
 -  `...rest`: additional files to create

## Examples

Creates "fixture.json"
```shell
> touch fixture.json
```

Creates files a, b and c
```shell
> touch a b c
```
