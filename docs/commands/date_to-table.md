Print the date in a structured table.

## Signature

```> date to-table ```

## Examples

Print the date in a structured table.
```shell
> date to-table
```

Print the date in a structured table.
```shell
> date now | date to-table
```

Print the date in a structured table.
```shell
>  '2020-04-12 22:10:57 +0200' | date to-table
```
