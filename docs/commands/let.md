Create a variable and give it a value.

## Signature

```> let (var_name) (initial_value)```

## Parameters

 -  `var_name`: variable name
 -  `initial_value`: equals sign followed by value

## Examples

Set a variable to a value
```shell
let x = 10
```

Set a variable to the result of an expression
```shell
let x = 10 + 100
```
